## Image Optimization Tips

1. **Format**: Use WebP for modern browsers with JPG/PNG fallbacks
2. **Compression**: Compress images to reduce file size (use tools like TinyPNG)
3. **Responsive**: Consider providing multiple sizes for different screen resolutions
4. **Alt Text**: Always include descriptive alt text in the HTML

## Photography Tips for Lawn Care

- Shoot during golden hour (early morning or late afternoon)
- Include before/after shots of actual work
- Show equipment in action
- Capture close-ups of lawn quality
- Include wide shots of entire properties
- Ensure lawns are freshly mowed with visible stripes
- Clean up any debris before photographing
